# solarman-rs

Rust utilities for communicating with Solarman V5 based solar inverter data logging sticks.

This repository consists of the following crates:
- [`solarman-tokio`](solarman-tokio) - high-level Tokio-based async client
- [`solarman-protocol`](solarman-protocol) - low-level bare implementation of the Solarman V5 protocol
